        How to read this dashboard

        1. Start with Data Health. If the status is not OK, be cautious about interpreting the scores.
        2. Read the Plain-English Interpretation before diving into the numbers to get the market narrative.
        3. Use Scores to identify the underlying drivers.
        4. Use Key Divergences to spot early cracks or leadership changes underneath the surface.
        5. Check the Matrix for today's visual regime position, and History to see how momentum is changing.

        Methodology & Limitations

        ASIC / Networking Rotation: This score does not prove GPU demand is falling. It measures whether the market is rotating toward ASIC, networking, and scale-out infrastructure relative to pure NVIDIA GPU beta.

        Breadth & Stress: Composite metrics measuring underlying market participation and high-beta drawdowns respectively.

        Note: This is a market-implied tool measuring relative momentum, not fundamental truth. It is not standalone investment advice.
